# Certificate of Enrollment (COE) Templates

Place your COE template images here. These are used by the AI to verify submitted Certificates of Enrollment.

## Template Files Expected

- `GENERAL_COE_FORMAT 1.jpg` - Standard COE format
- `GENERAL_COE_FORMAT 2.jpg` - Alternative COE format

## Format Guidelines

- **File Types**: JPG, PNG, PDF
- **Resolution**: Minimum 300 DPI
- **Content**: Should include typical COE elements:
  - School header/logo
  - Student information section
  - Enrollment details (semester, year, course)
  - Official signatures and seals
  - Date issued

## Privacy Note

⚠️ Ensure all templates are anonymized (remove real student names, IDs, etc.)
