# School ID Reference Samples

This directory contains reference samples of various school IDs for format validation.

## 🏫 School ID Types
- University student IDs
- High school IDs  
- Elementary school IDs
- Government school formats
- Private school formats

## 📋 What to Include
- Different card dimensions and layouts
- Various photo placement styles
- Different ID number formats
- School seal and logo variations
- Lamination and security features

## 🔒 Privacy Protection
- Use sample IDs or anonymized versions
- Blur or remove actual student photos
- Redact real student information
- Use placeholder data when possible

## 📝 File Naming
- `{school_type}_{school_name}_{year}.{ext}`
- Example: `university_tcu_student_id_2024.jpg`
- Example: `high_school_sample_id_2023.png`
