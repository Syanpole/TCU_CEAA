# Transcript of Records (TOR) Templates

Place your TOR template images here. These are used by the AI to verify submitted Transcripts of Records.

## Template Files Expected

- `TOR_TEMPLATE 1.jpg` - Standard TOR format

## Format Guidelines

- **File Types**: JPG, PNG, PDF
- **Resolution**: Minimum 300 DPI
- **Content**: Should include typical TOR elements:
  - School header/logo
  - Student information
  - Course listings with grades
  - Academic standing/GPA
  - Official signatures and seals
  - Registrar stamp
  - Date issued

## Privacy Note

⚠️ Ensure all templates are anonymized (remove real student names, IDs, grades, etc.)
