# Reference Document Samples

This directory contains official document samples for AI comparison and validation.

## 📄 Birth Certificates
- Place official PSA birth certificate samples here
- Include different formats and regional variations
- Examples: `psa_birth_cert_2024.pdf`, `civil_registry_sample.pdf`

## 🔒 Security Notes
- Ensure proper permissions before storing official documents
- Remove or redact sensitive personal information
- Use watermarked or sample documents when possible
- Consider using anonymized official templates

## 📋 Naming Convention
- `{issuing_authority}_{document_type}_{year}.{extension}`
- Example: `psa_birth_certificate_2024.pdf`
- Example: `civil_registry_birth_cert_2023.jpg`

- Regularly update samples with latest formats
- Check for new security features or design changes
