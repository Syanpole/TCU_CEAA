## Color Scheme

### Primary Colors
- **Purple Gradient**: `#667eea` → `#764ba2` (Header)
- **Success Green**: `#28a745` (Submit buttons, checkmarks)
- **Edit Purple**: `#764ba2` (Edit button)
- **Warning Orange**: `#f59e0b` (Lock icons, warnings)

### Secondary Colors
- **Background Overlay**: `rgba(0, 0, 0, 0.6)`
- **Card Background**: `#ffffff`
- **Section Background**: `#f8f9fa`
- **Border**: `#e0e0e0`
- **Disabled Input**: `#f5f5f5`

## Animations

### Fade In (Form Steps)
```css
@keyframes fadeIn {
  from {
    opacity: 0;
    transform: translateY(10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
Duration: 0.3s
```

### Scale In (Confirmation Dialog)
```css
@keyframes scaleIn {
  from {
    transform: scale(0.9);
    opacity: 0;
  }
  to {
    transform: scale(1);
    opacity: 1;
  }
}
Duration: 0.2s
```

### Spin (Loading)
```css
@keyframes spin {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}
Duration: 1s (continuous)
```

## Responsive Breakpoints

### Desktop (> 768px)
```
┌─────────────────────────────────────┐
│  Modal: 900px max-width, centered   │
│  3-column grid for form fields      │
│  All progress steps visible         │
└─────────────────────────────────────┘
```

### Mobile (≤ 768px)
```
┌───────────────────────┐
│  Full-screen modal    │
│  Single-column grid   │
│  Progress steps wrap  │
│  Larger touch targets │
└───────────────────────┘
```
