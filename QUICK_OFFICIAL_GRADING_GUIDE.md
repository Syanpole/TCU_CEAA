## 💰 Allowance Eligibility

### Merit Incentive (₱5,000)
```
Required: GWA ≤ 1.6 (87% or higher)
Eligible: 1.0, 1.1, 1.2, 1.3, 1.4, 1.5, 1.6
```

### Basic Allowance (₱5,000)
```
Required: GWA ≤ 1.9 (80% or higher)
Eligible: 1.0, 1.1, 1.2, 1.3, 1.4, 1.5, 1.6, 1.7, 1.8, 1.9
```

### Maximum Total
```
₱10,000 (Merit + Basic)
Get GWA 1.6 or better!
```

## 📊 Important Changes

### What Changed from Before:

**Eligibility Thresholds**:
```
Old Merit:  GWA ≤ 1.75
New Merit:  GWA ≤ 1.6   ← MORE STRICT

Old Basic:  GWA ≤ 2.74
New Basic:  GWA ≤ 1.9   ← MORE STRICT
```

**Impact on Common Grades**:
```
GWA 1.75:
  Before: ✅ Merit Eligible
  Now:    ❌ Basic Only (85%)

GWA 2.0:
  Before: ✅ Basic Eligible
  Now:    ❌ Not Eligible (79.5%)

GWA 2.5:
  Before: ❌ Not Eligible
  Now:    ❌ Not Eligible (same)
```
